Split Panels

Split panels display two adjacent panels, allowing the user to reposition them.

Initial Position

To set the initial position, use the position attribute. If no position is provided, it will default to 50% of the available space.

Initial Position in Pixels

To set the initial position in pixels instead of a percentage, use the position-in-pixels attribute.

Vertical

Add the vertical attribute to render the split panel in a vertical orientation where the start and end panels are stacked. You also need to set a height when using the vertical orientation.

Disabled

Add the disabled attribute to prevent the divider from being repositioned.

Min & Max

To set a minimum or maximum size of the primary panel, use the --min and --max custom properties. Since the secondary panel is flexible, size constraints can only be applied to the primary panel. If no primary panel is designated, these constraints will be applied to the start panel. This examples demonstrates how you can ensure both panels are at least 150px using --min, --max, and the calc function.
