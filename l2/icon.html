Icons are symbols that can be used to represent various options within an application.

FontAwesome

This will register the Font Awesome Free library using the jsDelivr CDN. This library has three variations: regular, solid and brands A mutator function is required to set the SVGs fill to currentColor. Icons in this library are licensed under the Font Awesome Free License. Some of the icons that appear on the Font Awesome website require a license and are therefore not available in the CDN
